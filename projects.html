Aetriks - Keylogger

Aetriks is a keylogger designed for educational purposes. It demonstrates how keystrokes can be securely captured and logged. The project includes a detailed contribution guide available on GitHub. Pull Request for contributions are welcome.

Aetriks Relay Server

The Aetriks Relay Server is a companion project for Aetriks. It acts as a secure relay server for transmitting logged data to a centralized location. Check out the repository for setup instructions and contribution guidelines. Pull Request for contributions are welcome.
